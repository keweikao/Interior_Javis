## 一、標準施工順序（工序流程）

### 全室裝修標準流程

```
保護工程 → 拆除工程 → 水電工程（配管線）→ 泥作工程（砌牆/防水/整平/地磚）
→ 防水工程（試水測試）→ 木作工程（天花板/隔間/造型）
→ 油漆工程（批土/底漆/面漆）→ 壁紙工程（如適用）
→ 地板工程（木地板/PVC/SPC）
→ 系統櫃/廚具安裝 → 空調設備安裝 → 門窗工程
→ 衛浴設備安裝 → 燈具/五金安裝 → 清潔工程（粗清→細清）
```

### 依屋況分類的施工順序

| 屋況 | 施工順序特點 |
|------|------------|
| **新成屋** | 無需拆除，從保護→水電→泥作開始 |
| **毛胚屋** | 從水電管線配置開始，需全面泥作 |
| **中古屋（10-20年）** | 需局部拆除，重點在水電更新、防水重做 |
| **老屋翻新（20年+）** | 全面拆除→結構補強→水電全面更新→防水→全室重做 |

### 工序間的硬性先後關係

```
拆除 ──→ 水電配管 ──→ 泥作封管 ──→ 防水 ──→ 貼磚
                 ↘                    ↗
                  泥作砌牆 ──→ 粉光 ──→ 防水（浴室）

木作天花板 ──→ 油漆批土 ──→ 油漆面漆 ──→ 壁紙（如適用）──→ 燈具安裝

地面整平 ──→ 地板鋪設 ──→ 收邊/矽利康收邊

系統櫃丈量（確認淨高）──→（工廠製作 2-4 週）──→ 現場安裝
```

## 二、各工種標準工項清單

|------|------|------|:---:|

## 三、工序相依規則（If-Then Rules）

### 關卡 1：工序連動（漏項檢查）

#### 拆除類

| 如果報價有... | 則必須有... | 嚴重度 | 說明 |
|-------------|-----------|:---:|------|
| 拆除天花板 | 天花板重做 或 油漆修補 | 🔴 | 拆了天花不可能不做（除非裸天花工業風） |
| 拆除地磚 | 地面整平 + 防水 | 🔴 | 拆除後高低差必須處理，見底後需防水 |
| 拆除壁磚 | 壁面粉光 或 重新貼磚 | 🔴 | 拆除後牆面裸露 |
| 拆除隔間牆 | 地面修補 + 天花板修補 | 🔴 | 牆拆除後地面天花有缺口；注意乾式/濕式拆法不同 |
| 拆除舊門窗 | 新門窗安裝 + 泥作填縫 | 🔴 | 不可能留洞不補；老屋需特別注意防水 |
| 任何拆除工程 | 垃圾清運 | 🔴 | **最常漏報！** |
| 任何拆除工程（大樓） | 保護工程 | 🟡 | 大樓施工幾乎必備；需確認保護材質 |

#### 泥作/防水類

| 如果報價有... | 則必須有... | 嚴重度 | 說明 |
|-------------|-----------|:---:|------|
| 新砌磚牆 | 粉光 + 油漆（或封板） | 🔴 | 裸磚不能住；確認是否有掛重物需補強 |
| 浴室拆除/翻新 | 防水工程（幾道需確認） | 🔴 | 浴室必做防水 |
| 防水工程 | 試水測試（通常包含在防水報價內） | 🔴 | 驗證防水有效的唯一方法 |
| 浴室貼磁磚 | 防水工程（先做） | 🔴 | 防水必須在貼磚之前 |
| 換窗 | 窗框泥作填縫 + 防水（建議打針預防） | 🔴 | 否則漏水 |

#### 水電類

| 如果報價有... | 則必須有... | 嚴重度 | 說明 |
|-------------|-----------|:---:|------|
| 冷氣安裝 | 洗洞 + 排水管（or 排水器）+ 220V 迴路 | 🔴 | 冷氣三要素 |
| 全熱交換器 | 洗洞（2-4 個） | 🔴 | 進排氣需穿牆 |
| 廚房翻新 | 瓦斯管線檢查/更新 | 🟡 | 安全考量 |
| 電熱水器安裝 | 專用迴路（30A以上） | 🔴 | 功率大需獨立迴路 |
| IH 爐安裝 | 220V 專用迴路 | 🔴 | 功率大需獨立迴路 |
| 馬桶移位 | 糞管移位 + 地板墊高 | 🔴 | 排汙管需要洩水坡度 |
| 新增插座（廚房） | 獨立迴路（≥2 迴路） | 🟡 | 廚房電器功率高 |
| 水電配管 | 泥作封管修補（確認管材：EMT or 一般） | 🔴 | 封管後極難變動 |
| 消防感知器需移位 | 消防感知器移位工程 | 🔴 | 確認由誰施工（消防 or 水電） |
| 消防灑水頭需移位 | 消防管線修改 | 🔴 | 確認由誰施工（消防 or 水電） |

#### 木作類

| 如果報價有... | 則必須有... | 嚴重度 | 說明 |
|-------------|-----------|:---:|------|
| 木作天花板 | 油漆（批土+底漆+面漆）或 壁紙 | 🔴 | 板材需要上漆或貼壁紙 |
| 木作門片 | 五金（鉸鏈+把手+門鎖），確認是否需隔音條 | 🟡 | 五金常另計但不可缺 |

### 關卡 3：數量合理性檢查

| 工項 | 合理比例/公式 | 異常判定 |
|------|-------------|---------|
| **油漆面積** | 地坪 × 2.5 ~ 4.0（含牆面+天花） | < 2.0x 或 > 5.0x 異常 |
| **磁磚面積（地）** | ≈ 實際鋪設面積 × 1.05~1.10（含損耗），看磁磚種類級數 | 損耗率 > 15% 異常 |
| **磁磚面積（壁）** | 浴室：(周長 × 高度) × 1.08 | - |
| **電線迴路數** | 基本 6~8 迴路 + 廚房 2~3 + 冷氣每台 1；7~8 個插座一迴路為極限 | 20 坪以上少於 8 迴路異常 |
| **插座數量** | 每坪 2~3 個（含既有） | 低於每坪 1 個異常 |
| **冷氣噸數** | 每坪 0.4~0.5 噸（一般）/ 0.5~0.7 噸（西曬/頂樓） | 偏差 > 30% 異常 |
| **垃圾清運車數** | 全室拆除：每 10 坪約 2 車 | 20 坪全拆 < 2 車異常 |
| **防水面積（浴室）** | 地面全做 + 牆面淋浴區（H=90/130/全部） | 面積明顯少於地坪 |
| **批土面積** | ≈ 油漆面積（通常等同） | 油漆有報但批土沒報 |
| **木地板** | ≈ 實際面積 × 1.05~1.08（含損耗） | 損耗率 > 12% 異常 |

## 五、報價單常見格式（欄位結構）

### 典型報價單欄位

```
| 項次 | 工程項目 | 規格說明 | 單位 | 數量 | 單價 | 複價(小計) | 備註 |
```

### 常見分類方式

報價單通常按工種分大類，每個大類下列出細項：

```
一、拆除工程
  1. 天花板拆除          坪    12    1,200    14,400
  2. 地磚拆除            坪    25    1,500    37,500
  3. 垃圾清運            車     3    8,000    24,000
                                    小計     75,900
二、水電工程
  1. 全室重新配線        式     1   120,000   120,000
  2. 開關插座增設        個    15    2,000    30,000
  ...                               小計    150,000

（各工種依此類推）

                                    總計    XXX,XXX
                                    稅金（5%）XX,XXX
                                    合計    XXX,XXX
```

### 常見計價單位

| 單位 | 使用時機 |
|------|---------|
| 坪 | 面積類：油漆、地板、天花板、拆除 |
| 才 | 門窗玻璃（1才 = 30cm × 30cm） |
| 尺 | 線性：系統櫃、踢腳板、窗簾盒 |
| 式 | 整體包含：防水、清潔、訂製品 |
| 個/組 | 單品：插座、洗洞、衛浴設備 |
| 台 | 設備：冷氣、抽風機 |
| 樘 | 門窗：一扇門或一扇窗 |
| 車 | 垃圾清運 |
| 工 | 人工：粗清、細清 |
| 米 | 管線類：冷媒管、排水管 |
| 迴路 | 電線迴路 |
| M² | 特殊漆（珪藻土等）、石膏磚隔間 |

## 六、報價單常見問題（紅旗警示）

### 格式面紅旗

| 警示 | 說明 | 風險 |
|------|------|------|
| 大量「一式」計價 | 無法驗證數量合理性 | 🔴 高 |
| 無規格說明 | 材料品質無從驗證 | 🔴 高 |
| 無單價只有總價 | 無法分辨工料比 | 🟡 中 |
| 工種分類混亂 | 不同工種混在一起 | 🟡 中 |
| 無備註/施工說明 | 施工範圍模糊 | 🟡 中 |

### 內容面紅旗

| 警示 | 說明 | 風險 |
|------|------|------|
| 報價金額過低 | 可能故意壓低，後續追加 | 🔴 高 |
| 工項過少（相對工程規模） | 可能大量漏項 | 🔴 高 |
| 缺少垃圾清運 | 幾乎所有工程都需要 | 🟡 中 |
| 缺少保護工程（大樓） | 管委會可能罰款 | 🟡 中 |
| 浴室翻新但無防水 | 致命漏項 | 🔴 高 |
| 有拆除但無整平 | 後續工程基礎問題 | 🔴 高 |
| 同工項重複計價 | 同一項目在不同工種中重複出現 | 🔴 高 |
| 材料規格模糊 | 只寫「磁磚」不寫品牌/型號/尺寸 | 🟡 中 |
| 缺少壁紙工程 | 有壁紙設計但報價未列 | 🟡 中 |

### 常見報價陷阱（防呆重點）

| 陷阱類型 | 說明 | 防呆建議 |
|---------|------|---------|
| **「一式」灌水** | 用「一式」包含大量工項，無法逐項驗證 | 要求拆分成明細 |
| **故意低報** | 壓低總價搶案，施工後大量追加 | 對比市場行情，過低需警示 |
| **隱藏必要項** | 刻意不報「垃圾清運」「保護工程」等必要項 | 自動檢查必要項是否存在 |
| **雙重計價** | 同一工項在不同類別重複出現 | 交叉比對工項名稱 |
| **材料不標規格** | 只寫品名不寫型號，實際用低等材料 | 提醒規格欄必須填寫 |
| **工程範圍不明** | 「油漆」但不說哪些區域 | 要求標明施作範圍 |
