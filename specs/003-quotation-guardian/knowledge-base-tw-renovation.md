## 一、標準施工順序（Standard Construction Sequence）

### 1.1 完整施工流程

```
保護工程 → 拆除工程 → 放樣 → 水電配管（粗配）→ 冷氣配管
→ 泥作（砌磚/防水/打底/貼磚）→ 木作（天花板/隔間/櫃體）
→ 油漆（批土/底漆/面漆）→ 系統櫃安裝 → 地板鋪設
→ 水電安裝（細裝：開關面板/燈具/衛浴設備）→ 冷氣安裝
→ 清潔 → 驗收 → 補漆/收尾 → 完工
```

### 1.2 階段與工期參考

| 階段 | 工項 | 參考工期 |
|------|------|----------|
| 第一階段 | 保護工程 | 1-2 天 |
| 第二階段 | 拆除工程 | 3-5 天 |
| 第三階段 | 水電粗配 + 冷氣配管 | 1-2 週 |
| 第四階段 | 泥作（砌磚/防水/打底/貼磚） | 2-3 週 |
| 第五階段 | 木作（天花板/隔間/櫃體） | 2-4 週 |
| 第六階段 | 油漆工程 | 1-2 週 |
| 第七階段 | 系統櫃/廚具安裝 | 1-2 週 |
| 第八階段 | 地板鋪設 | 3-7 天 |
| 第九階段 | 水電細裝 + 冷氣掛機 + 燈具 | 3-5 天 |
| 第十階段 | 清潔 + 驗收 | 2-3 天 |

> **一般 20-35 坪住家總工期**：新成屋約 3-4 個月，中古屋約 6-8 個月，老屋全翻約 8-12 個月。

### 1.3 施工大原則

- **由上而下、由內而外、由濕而乾**
- 拆除順序：由上而下、由內而外、由木而土，大門最後拆
- 後做的工項不可影響前做的工項
- 不可搶快施作，必須給足養護時間（如水泥砂漿養護至少 7 天）

| **專用迴路** | 新增大功率電器（冷氣/烘衣機/電器櫃） | 2,500-4,500/迴路 |

### 3.2 常見漏項（Warning：容易遺忘）

| 漏項 | 觸發條件 | 說明 |
|------|----------|------|
| **冷氣配管排水** | 冷氣位移或新裝 | 銅管、排水管、控制線 |
| **冷氣排水器** | 吊隱式冷氣 | 靜音排水器 |
| **維修孔** | 吊隱式冷氣/天花板內管線 | 供日後檢修使用 |
| **窗簾盒** | 天花板工程 | 常忘記預留 |
| **踢腳板/收邊條** | 地板工程 | 地板與牆面交接 |
| **門檻收邊** | 地板工程，不同區域交接 | 高低差收邊 |
| **無電梯搬運加價** | 無電梯公寓 3F 以上 | 每層加價 15%-25% |
| **電箱更換** | 老屋（屋齡 >20 年） | 總開關箱不符現行標準 |
| **漏電斷路器** | 浴室/廚房迴路 | 法規要求 |
| **弱電配線** | 網路/電視/對講機 | 常只算強電忘記弱電 |
| **燈具安裝工資** | 業主自購燈具 | 安裝費另計 |
| **清潔費** | 所有案件 | 粗清+細清 |
| **開關插座面板** | 水電工程 | 有時只算線不算面板 |
| **矽利康收邊** | 浴室/廚房/窗框 | 收尾工項容易漏 |
| **油漆天花板** | 油漆工程 | 只算牆面忘記天花板 |

### 3.3 報價陷阱模式

| 陷阱類型 | 說明 | 辨識方法 |
|----------|------|---------|
| **一式模糊計價** | 用「一式」包大項，不明確定義範圍 | 要求拆分細項 |
| **低價搶單** | 刻意漏報必要項目壓低總價 | 比對本清單 |
| **重複計價** | 同一空間或工項重複收費 | 核對空間範圍 |
| **材料規格模糊** | 不標示品牌、規格、產地 | 要求補充 |
| **人工材料不分** | 連工帶料無法拆分比較 | 要求分列 |
| **只報上櫃不報下櫃** | 系統櫃故意少報 | 核對設計圖 |
| **不含搬運費** | 底部備註寫不含搬運 | 檢查備註欄 |

## 四、坪數與工項數量經驗公式

### 4.1 油漆面積估算

| 公式 | 適用場景 | 說明 |
|------|---------|------|
| 地坪 × 2.7 | 四面牆面積（不含天花板） | 最常用公式 |
| 地坪 × 2.0 | 含天花板，無隔間的大空間 | 粗略估算（下限） |
| 地坪 × 2.5 | 含天花板，一般格局 | 中間值 |
| 地坪 × 3.7 | 含天花板，有多隔間的居家 | 經驗值（上限） |
| （長+寬）× 高 × 2 | 精算四面牆 | 精確計算 |
| 地坪面積 | 天花板面積 | 天花板≈地坪 |

> **範例**：室內 30 坪住家（三房兩廳），油漆面積約 30 × 3.0 = 90 坪（含天花板和牆面）。若只報 40 坪顯然不足。

### 4.2 電線用量估算

| 項目 | 經驗值 |
|------|--------|
| 一般住家插座數量 | 每坪 2-3 個 |
| 專用迴路 | 冷氣每台 1 條、廚房電器櫃 1-2 條、浴室暖風機 1 條 |
| 全室電線重拉（30 坪） | 約 60,000-120,000 元 |

### 4.3 泥作面積估算

| 項目 | 經驗值 |
|------|--------|
| 浴室壁磚面積 | 浴室地坪 × 4-5（四面牆） |
| 浴室地磚面積 | 浴室地坪面積 |
| 廚房壁磚（一字型） | 長度 × 高度（通常 60-80cm） |

### 4.4 木作天花板面積

| 項目 | 經驗值 |
|------|--------|
| 平釘天花板 | ≈需施作空間的地坪面積 |
| 間接照明長度 | 量測天花板邊緣周長 |
| 包樑長度 | 量測各樑的長度總和 |

### 4.5 系統櫃數量

| 項目 | 經驗值 |
|------|--------|
| 衣櫃 | 每間臥室 6-8 尺 |
| 鞋櫃 | 3-5 尺 |
| 電視櫃 | 5-8 尺 |
| 廚房上下櫃 | 依廚房長度 |

## 五、工序相依關係（Dependency Rules）

### 5.1 硬性相依（MUST：一定要有）

| 前置工項 | → 必須伴隨的工項 | 嚴重度 | 說明 |
|----------|-----------------|--------|------|
| 拆除地磚 | → 地面整平（泥作） | Critical | 拆除後地面高低不平 |
| 拆除壁磚 | → 壁面整平（泥作） | Critical | 拆除後壁面坑洞 |
| 浴室拆除 | → 防水工程 | Critical | 必須重做防水 |
| 防水工程 | → 48 小時試水 | Critical | 驗證防水品質 |
| 浴室防水 | → 洩水坡施作 | Critical | 排水必須 |
| 更換鋁窗 | → 窗框泥作填縫 | Critical | 窗框與牆面交接 |
| 全熱交換器 | → 洗洞（鑽孔） | Critical | 外牆進排氣孔 |
| 排油煙機管線 | → 洗洞（鑽孔） | Critical | 排煙管穿牆 |
| 水電配管打壁 | → 管溝泥作修補 | Critical | 打壁後填補 |
| 拆除隔間 | → 地面修補整平 | Critical | 隔間拆除留下溝痕 |
| 馬桶移位 | → 排水管重配 + 墊高地面 | Critical | 需糞管坡度 |
| 冷氣安裝 | → 專用迴路 | Critical | 大功率需獨立迴路 |
| 烘衣機安裝 | → 專用迴路 + 排水 | Critical | 220V 獨立迴路 |
| 電器櫃 | → 專用迴路 | Critical | 大功率電器集中 |

### 5.2 強烈建議（WARNING：高機率需要）

| 前置工項/條件 | → 建議搭配工項 | 嚴重度 | 說明 |
|--------------|---------------|--------|------|
| 吊隱式冷氣 | → 維修孔 | Warning | 日後檢修必要 |
| 吊隱式冷氣 | → 排水器 | Warning | 靜音排水 |
| 天花板施作 | → 窗簾盒 | Warning | 事後補做很麻煩 |
| 天花板施作 | → 燈具出線口 | Warning | 預留燈具位置 |
| 地板鋪設 | → 踢腳板/收邊條 | Warning | 牆面與地板交接 |
| 不同地板材交接 | → 門檻收邊條 | Warning | 過渡收邊 |
| 老屋（>20 年） | → 全室電線重拉 | Warning | 線路老化 |
| 老屋（>20 年） | → 給水管更換 | Warning | 管路老化鏽蝕 |
| 老屋（>20 年） | → 總開關箱更換 | Warning | 不符現行法規 |
| 無電梯 3F 以上 | → 搬運加價費 | Warning | 人力搬運 |
| 任何拆除 | → 垃圾清運費 | Warning | 廢棄物處理 |
| 所有案件 | → 保護工程 | Warning | 保護未施工區域 |
| 所有案件 | → 清潔費 | Warning | 完工清潔 |

### 5.3 工序時序規則（不可顛倒）

```
1. 保護工程 → 在所有施工前完成
2. 拆除 → 在任何新建工程前完成
3. 水電粗配 → 在泥作封閉前完成（管線埋入牆壁）
4. 冷氣配管 → 在天花板封板前完成
5. 鋁窗立框 → 在泥作填縫前完成
6. 泥作砌磚 → 在粉光前完成
7. 泥作防水 → 在貼磚前完成
8. 泥作打底 → 在貼磚前完成
9. 木作天花板 → 在油漆前完成
10. 木作隔間 → 在油漆前完成
11. 油漆批土 → 在面漆前完成
12. 系統櫃安裝 → 在油漆後（避免弄髒櫃體）
13. 地板鋪設 → 在大部分工程完成後（避免損傷）
14. 水電細裝（開關面板/燈具） → 在油漆後
15. 清潔 → 最後進行
```

## 六、現場條件對報價的影響

### 6.1 樓層與電梯

| 條件 | 影響 | 加價幅度 |
|------|------|---------|
| 有電梯大樓 | 基準價格 | — |
| 無電梯 1-2F | 影響極小 | 0-5% |
| 無電梯 3F | 搬運加價 | 10-15% |
| 無電梯 4F | 搬運加價 | 15-20% |
| 無電梯 5F 以上 | 搬運加價+部分建材限制 | 20-30% |
| 電梯過小（無法放大型建材） | 需走樓梯搬運 | 同無電梯計算 |

### 6.2 屋齡

| 屋齡 | 裝潢費用/坪 | 必須注意事項 |
|------|------------|-------------|
| 新成屋（0-5 年） | 6-8 萬 | 管線新，主要做設計裝飾 |
| 中古屋（10-20 年） | 8-10 萬 | 部分管線可能需更新 |
| 老屋（20-30 年） | 8-12 萬 | 水電管線建議全面更新 |
| 老屋（30-40 年） | 10-14 萬 | 需檢查結構安全、全面翻新 |
| 危老（>40 年） | 12 萬以上 | 可能需結構補強 |

> **老屋基礎工程佔比**：泥作+水電約佔整體預算 50%-60%。

### 6.3 挑高

| 天花板高度 | 影響 |
|-----------|------|
| 標準（280-300cm） | 基準 |
| 挑高（300-360cm） | 油漆/泥作壁面面積增加、鷹架費 |
| 超挑高（>360cm） | 需搭設鷹架，費用顯著增加 |

### 6.4 其他現場條件

| 條件 | 對報價的影響 |
|------|-------------|
| 潮濕/漏水 | 需加做防水/抓漏，費用增加 |
| 巷道狹窄 | 大型建材進場困難，搬運加價 |
| 停車不便（步行>20m） | 搬運步行距離加價 |
| 施工期間有人住 | 需分區施工、保護更多、工期拉長 |
| 社區管委會管制 | 限制施工時間、噪音管制、保證金 |
| 頂樓加蓋 | 需注意防水、隔熱、結構 |
| 地下室 | 通風、搬運、防潮 |

### 6.5 地區價差

| 地區 | 相對於雙北的價差 |
|------|-----------------|
| 台北市/新北市 | 基準 |
| 桃園/新竹 | -5% ~ -10% |
| 台中 | -10% ~ -15% |
| 台南/高雄 | -15% ~ -20% |
